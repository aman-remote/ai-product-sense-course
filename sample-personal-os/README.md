# Sample Personal OS

A ready-to-use Personal OS so you can run any lesson without building one first — and without pasting confidential work data. Everything here is fictional (a made-up PM named Sam at a fictional B2B product called "Cadence").

Point your agent at this folder whenever a lesson says "use your Personal OS" or "your real files" and you don't have one (or can't use work data).

```
sample-personal-os/
├── GOALS.md          ← what Sam is trying to achieve this quarter
├── BACKLOG.md        ← a messy brain-dump to triage
├── AGENTS.md         ← agent instructions / working style
├── Tasks/            ← individual task files with priority + status
└── Knowledge/
    └── voice-samples/ ← writing samples for voice-training lessons
```
