Bottom line. The course is structurally sound: 19 of 20 lessons teach a clear concept with a falsifiable hands-on exercise, and the strongest lessons (06, 07, 16) actively de-magic themselves. Five issues converged across multiple personas and are worth fixing. The single highest-leverage fix: a complete sample-personal-os/ fixture already exists in the repo, but no lesson references it by name, so the no-data / mid-course student never finds the lifeboat that was built for them.
Audience calibration. The real student is a ChatGPT/Claude-literate PM who can install an app and run a setup command — not a true novice. The beginner persona (Anna) is a stress-test floor. Her friction is recorded as data; her fixes are applied only where they help everyone (silent-failure bugs, fallbacks), not where they'd patronize the target user (making required steps optional, anxiety framing). Five LLM role-plays are directional, not proof.

Findings, ranked by persona convergence

P0The sample-personal-os/ fixture is invisible.

Convergence: Marcus (hard), Priya, Kenji. A complete privacy-safe fixture (GOALS, BACKLOG, AGENTS, Tasks/, Knowledge/voice-samples/) sits in the repo root, purpose-built for the no-data student. No lesson names it. Lessons 14/15/17/19 each reinvent a thinner throwaway instead.

Fix: add one line to every lesson that needs a project folder or voice samples: "No folder of your own, or can't use work data? Point your agent at sample-personal-os/."

P0Cross-reference lesson numbers are stale.

Convergence: Kenji (got disoriented), plus known renumber debt. Bodies still say "Lesson 17" / "Lesson 18" / "Lesson 16" pointing at the pre-renumber order (26 inline mentions). 14→16, 15→17, 16→19, 17→18 don't line up.

Fix: remap all 26 inline Lesson N references to the new numbering.

P0Lesson 14 super-stretch restructures real files with no backup note.

Convergence: Priya (silent-failure), echoes Marcus privacy. "Do it now — restructure" can move/lose real work files silently, while Lesson 11 (the other destructive lesson) is locked down with reversible-first warnings.

Fix: copy Lesson 11's safety note — commit/copy first, reversible operations, confirm before moving.

P1Lesson 13 is overloaded and its memory half is narrated, not demoed.

Convergence: Anna (highest quit-risk), Dev (oversold), Priya (bloated, two magic moments). The search half is falsifiable and strong; the MEMORY.md "dream state" half is described out loud and oversold as "the same architecture" as ChatGPT memory (which the lesson itself later says is NOT RAG).

Fix: soften the memory-half overclaims (drop "exactly like you," reconcile the "same architecture" line), label the dream-state as a setup-it-yourself pipeline not a watch-it-now demo, and add the sample-personal-os/ fallback for the clone step.

P1Unsourced fake-precision numbers.

Convergence: Dev (primary), Anna (jargon-without-gloss overlap). "roughly 10-15 tools" (repeated 4x), "90% planning, 10% coding", "scales to hundreds" — all asserted, none sourced. Skeptic disengages exactly here.

Fix: cut the fake precision or hedge it ("a small fixed set", "mostly planning"). Don't invent metrics.

P2Idiom and locale friction for non-native readers.

Convergence: Kenji (primary), some overlap with Anna jargon. "get away with", "drop a line", "desire paths", "smart intern", "#1 tell", "pick your brain", "moat", "Wizard-of-Oz", "hive mind". Plus locale-bound demos: Spotify AI DJ and Granola (unavailable/unknown in Japan), Notion OAuth blocked by corporate SSO, "say no" voice samples that are US-direct.

Fix: gloss load-bearing idioms on first use; offer a globally-known demo alternative where the example is region-locked. Lower priority — the target audience clears these.

  WHERE IT BROKE: first instruction ("clone a repo") is the exact
  git/terminal action she fears; "grep", "cron", and two dense ASCII

  FIX APPLIED: added sample-personal-os/ fallback so no clone is
  required; softened memory-half overclaims; labeled dream-state as
  opt-in setup, not a live demo. (Idiom/gloss pass = P2, deferred.)
